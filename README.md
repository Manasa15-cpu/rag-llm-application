# RAG LLM PDF Question Answering Application

## Overview
This project is an end-to-end Retrieval-Augmented Generation (RAG) application built with Python and Streamlit.

## Features
- Upload PDF documents
- Extract text from PDFs
- Split text into chunks
- Prepare documents for AI-based question answering
- Simple Streamlit user interface

## Technologies Used
- Python
- Streamlit
- PyPDF2
- LangChain
- OpenAI API
- Git & GitHub

## How to Run

bash
pip install -r requirements.txt
streamlit run app.py


## Note
The AI question-answering feature requires a valid OpenAI API key with available API credits.
